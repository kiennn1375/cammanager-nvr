# CamManager ‚Äì Local PC-based NVR for IP Cameras
> Turn your Windows PC into a scalable, local NVR ‚Äî no cloud, no vendor lock-in.

CamManager is a Windows-based NVR (Network Video Recorder) software that turns a standard PC into a fully local video recording system for IP cameras.

Instead of relying on proprietary hardware NVRs with fixed storage limits and vendor lock-in, CamManager records video streams directly to local disks on your PC or server. Storage can scale freely by adding more drives, and your video data always stays under your control.

CamManager works with RTSP-compatible cameras from multiple manufacturers and does not require any cloud service or external account.

---

## Key Features
- PC-based NVR (no dedicated NVR hardware required)
- Records IP cameras directly to local storage
- Unlimited storage (limited only by available disk capacity)
- Supports RTSP cameras from multiple vendors
- Fully local operation (no cloud dependency)
- No vendor lock-in
- Designed for reliable 24/7 continuous recording

---

## Typical Use Cases
- Replacing hardware NVRs with a PC-based solution
- Mixed-brand IP camera environments
- Users who need flexible, expandable storage
- Privacy-focused and self-hosted setups
- Home labs, small businesses, and on-premise deployments
